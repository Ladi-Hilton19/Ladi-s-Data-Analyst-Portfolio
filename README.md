## [Exploratory Analysis for UK Road Safety Data (2010–2020)](https://github.com/Ladi-Hilton19/Leicestershire-Road-and-Accidents-Data) 

### Data Source

* UK Department for Transport – Road Safety Data Portal
* Dataset: STATS19 Road Accident and Casualty Data
* Format: CSV files containing accident and casualty records from 1979–2020

### Libraries & Technologies Used

* PySpark
* Spark SQL
* Hadoop Distributed File System (HDFS)
* Jupyter Notebook
* Python 3
* RDDs & DataFrames
* Pandas (supporting analysis)
* Data Visualization Libraries

### Project Highlights

Conducted a large-scale road safety analysis on over 20 million accident and casualty records for Leicestershire using PySpark and Spark SQL within a distributed computing environment. The analysis revealed a steady decline in accidents and casualties between 2010 and 2020, with Friday identified as the highest-risk day and Leicester City accounting for the highest concentration of incidents. Additional findings showed that single carriageways, daylight conditions, and fine weather were associated with the majority of reported accidents, while individuals aged 16–45 represented the largest casualty group.


## [Guided Learning Hours Analysis in UK Colleges (SAS Studio)](https://github.com/Ladi-Hilton19/Guided-Learning-Hours-across-UK-Colleges)

### Data Source

* UK Sixth Form and Further Education College datasets
* CSV datasets containing Guided Learning Hours (GLH) records across multiple UK regions and academic years

### Methods & Tools Used

* SAS Studio
* PROC IMPORT
* PROC CONTENTS
* PROC GLM
* PROC ANOVA
* PROC TTEST
* PROC UNIVARIATE
* Box-Cox Transformation
* Data Cleaning & Feature Engineering
* Statistical Modeling & Exploratory Data Analysis

### Project Highlights

Performed a statistical analysis of Guided Learning Hours (GLH) across UK Sixth Form and Further Education colleges using SAS Studio. The study identified college type and regional location as the strongest factors influencing GLH per learner, while institution size showed little statistical significance. Additional analysis revealed that Sixth Form Colleges consistently recorded higher GLH per learner compared to Further Education colleges across multiple academic years.
